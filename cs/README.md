## cs

### emoji_printer

Скрипт, принимающий в качестве аргументов символ фона (1), символ основного цвета (2) и слово (или несколько слов, кириллица). Печатает это слово сверху вниз, используя для цветов указанные символы. Неизвестные символы пропускает. Запускается командой "dotnet run emoji_printer.cs __ ЖЖ ваша фраза тут". Можно запустить "dotnet run emoji_printer.cs" без аргументов, напечатает слово "привет", оно не отобразится в консоли, но если скопировать и вставить туда, где отображаются emoji (мессенджеры) - корректно отобразит вывод.
