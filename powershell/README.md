## powershell

### find-cyrillic

Ищет в текущей директории и во всех вложенных файлы с кириллицей.

### find-cyrillic-explains

Ищет в текущей директории и во всех вложенных файлы с кириллицей и приводит первую кириллическую строку в качестве доказательства.

### tree

Копируется в необходимую папку, затем, когда исполняется - выводит дерево (структуру) проекта. Можно настроить выввод в консоль или файл. Можно настроить расширения файлов, которые будут выводиться и папки-исключения, которые будут проигнорированы.
